# 2 tipos de comunicação no backend:

### 2. REST (estado inicial do jogo)

    Criar jogo

    Entrar no jogo

    Buscar estado atual

    REST = estado inicial / refresh

### 3. WebSocket (jogo em tempo real)

    Enviar jogadas

    Receber atualizações

    Receber estado sincronizado

    WebSocket = tempo real
